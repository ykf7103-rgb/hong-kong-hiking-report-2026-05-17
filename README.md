# 香港短途靚景行山建議

公開網站：
https://ykf7103-rgb.github.io/hong-kong-hiking-report-2026-05-17/

這是一頁可公開瀏覽的 GitHub Pages 圖文報告，整理 2026-05-17 香港短途行山建議、泊車/落腳點、入口路線、短線走法、風險提示和真實相片。

目前內容：

- 40 個香港短途靚景地點。
- 每個地點最少 3 張真實相片，共 120 張圖片。
- 左方分組導覽，可點按地名即時跳到該區。
- 40 個地點入口、泊車、短線時間速查表。
- 分組包括：今日全港短線、屯門出發短線、九龍快到短線、自然呼吸短線。

最新新增：10 個可看大自然、呼吸新鮮空氣、步行 1 小時內的短線地點，包括迪欣湖、青衣自然徑、大埔滘、鳳園、嘉道理農場、北潭涌自然徑、獅子會自然教育中心、清水灣樹木研習徑、大埔海濱公園和大美督。
